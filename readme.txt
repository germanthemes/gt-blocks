Innovative Business Blocks for the new WordPress Editor.

== Description ==

With our flexible and innovative blocks for the new WordPress Editor, you can create complex layouts for your business website in just a few minutes.

= Included Blocks =

* GT Hero Image
* GT Portfolio
* GT Features
* GT Section
* GT Columns
* GT Grid Layout
* GT Image Card
* GT Multiple Buttons
* GT Icon
* GT Heading
* GT Button

= Documentation =

For detailed setup instructions, please see our [GT Blocks Documentation](https://germanthemes.de/en/docs/gt-blocks-documentation/)

= Development =

GT Blocks is developed on [Github](https://github.com/germanthemes/gt-blocks)

== Installation ==

1. Activate the plugin
2. Create a new post or page and use GT Blocks in the WordPress Block Editor
3. For detailed setup instructions, please see our [GT Blocks Documentation](https://germanthemes.de/en/docs/gt-blocks-documentation/)
